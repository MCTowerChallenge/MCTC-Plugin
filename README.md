# TowerChallenge
PaperMC Plugin that is designed for the Minecraft Tower Challenge event.

Developed by: Mystievous ([Github](https://github.com/Mystievous), [Twitter](https://twitter.com/mystievous))

## Winter 2022 Features

- [GUI system](src/main/java/io/github/mystievous/towerchallenge/gui)
- Event Function Handling
- Automatically assigning player teams
- [Questing System with NPCs](src/main/java/io/github/mystievous/towerchallenge/quests)
- Statistic Tracking for Bonus Prizes
- Fast Travelling between spawn and build area
- [Teleport History tracking](src/main/java/io/github/mystievous/towerchallenge/teleports)
- Message command that sends messages to the entire team
- Magic items that do various functions
- A compass to point you back to spawn, in any dimension
- Event timer that automatically stops at intermission and the end, and isn't disrupted by tick lag on the server

### Plugin Dependencies:

- [WorldGuard](https://dev.bukkit.org/projects/worldguard)
- [WorldEdit](https://dev.bukkit.org/projects/worldedit) (or [FastAsyncWorldEdit](https://www.spigotmc.org/resources/fastasyncworldedit.13932/))
- [NBT API](https://www.spigotmc.org/resources/nbt-api.7939/)
